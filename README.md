## Description

Session-based authentication with [NestJS](https://github.com/nestjs/nest) and [PassportJS](https://www.passportjs.org/)

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
